# 开发任务

1. 初始化Node 22 TypeScript CLI、环境变量校验、缓存和日志。
2. 建立分类、设施、来源记录和地铁线路关系的数据模型。
3. 实现HTML、PDF、XLSX/CSV官方来源适配器和人工快照导入。
4. 实现高德Web服务API客户端：限流、重试、分页、缓存和错误分类。
5. 实现官方记录的POI匹配、坐标补全、分类扫描和复核队列。
6. 实现去重、上海边界校验、等级证据校验和质量报告。
7. 导出Excel、CSV、GeoJSON、来源清单和复核工作簿。
8. 编写离线测试夹具，完成单区真实API试运行后再执行全量采集。
